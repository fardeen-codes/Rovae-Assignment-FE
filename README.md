# Product Catalog Application

A modern and responsive Product Catalog web application built using **React** and **Vite**.  
This project demonstrates component-based architecture, clean UI design, and scalable frontend structure.

---

## Tech Stack

- React (Functional Components + Hooks)
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- ESLint

---

## Features

- Product listing page
- Reusable UI components
- Responsive design (Mobile + Desktop)
- Clean folder structure
- Optimized build using Vite
- Modern UI with Tailwind CSS

cd <project-folder-name>
